Installation

Quickly get started testing your ARKit Projects.
1. Goto QAAR Portal

Create New Project And Add Test Case Questions.

2. Download QAAR.Framework

Drag QAAR Framework into your project.

3. Install MagicTouches

Setup the QAAR Custom App Class or install a custom sendEvent method in your existing custom AppClass.

4. Copy QAARJSON URL

Use the url when instantiating the QAARManager in code
